# Cloud-Native File Processing Platform

## Project Overview

This project documents my journey of designing and building a cloud-native file processing platform using Google Cloud.

Rather than following a tutorial, I'm approaching this like a real engineering project by documenting the planning, architecture decisions, implementation, testing, and lessons learned along the way.

## Why I'm Building This

One of the best ways I learn is by building.

As someone with a background in Microsoft 365 administration and automation, I wanted to expand my experience into cloud-native application development and serverless computing.

This project will help me gain hands-on experience with:

- Google Cloud Platform (GCP)
- Serverless Functions
- Event-Driven Architecture
- Cloud Storage
- Firestore
- IAM
- Cloud Logging
- Cloud Monitoring

## Project Goals

- Learn Google Cloud fundamentals
- Build a serverless application
- Understand event-driven architecture
- Document design decisions
- Develop a portfolio project that demonstrates cloud-native development skills
